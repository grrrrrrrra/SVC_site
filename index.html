Resolving Ambiguity in Embodied Instructions
via Semantic Valency Conflict

Daria Gitalova1, Alexey K. Kovalev1,2, Aleksandr I. Panov1,2

1MIPT, Moscow, Russia    2AIRI, Moscow, Russia

Paper PDF arXiv

Abstract

Natural language instructions in human–robot interaction often contain subtle ambiguities that hinder reliable interpretation. These arise when a single instruction can be understood in multiple ways, assigning conflicting semantic roles to objects, tools, or participants. To address this, we propose Semantic Valency Conflict (SVC) — a cognitively inspired, logit-free method for detecting ambiguity in robot-directed instructions. SVC identifies divergences in role assignments across alternative interpretations of a predicate using large language models to infer context-sensitive semantic frames.

Method

We define Semantic Valency Conflict as a condition where multiple interpretations of a lexeme activate incompatible semantic valency profiles in a given environment.
